01 Ringkasan Eksekutif

1.1 Overview

Dokumen ini merupakan laporan resmi implementasi fitur Navigasi Modular pada sistem Business Command Portal milik PT Utama Globalindo Cargo. Pengembangan ini mencakup restrukturisasi sistem navigasi sidebar dengan mengelompokkan seluruh fitur CRM ke dalam satu parent menu yang dapat di-expand/collapse, serta penambahan placeholder untuk Ticketing Module yang akan dikembangkan pada fase berikutnya.

02 Latar Belakang

2.1 Kondisi Sebelumnya

Business Command Portal sebelumnya memiliki struktur navigasi flat dimana seluruh menu ditampilkan secara langsung di sidebar tanpa pengelompokan.

Sidebar (Sebelum)
├── Overview
├── Lead Management
├── Lead Bidding
├── My Leads
├── Pipeline
├── Accounts
├── Sales Plan
├── Activities
└── Imports

2.2 Permasalahan yang Diidentifikasi

No Permasalahan Dampak
1 Navigasi tidak terstruktur User kesulitan menemukan menu yang relevan
2 Tidak ada pengelompokan modul Sulit membedakan fungsi antar menu
3 Tidak scalable Penambahan modul baru akan membuat sidebar terlalu panjang
4 Tidak ada visual hierarchy Semua menu terlihat setara tanpa konteks

2.3 Kebutuhan Bisnis

PT Utama Globalindo Cargo berencana mengembangkan Ticketing Module untuk menangani customer support dan internal ticketing. Untuk mengakomodasi pengembangan ini, diperlukan restrukturisasi navigasi yang memisahkan modul-modul secara jelas.

Catatan Terkait ERP:
  • ERP yang saat ini digunakan oleh UGC Logistics adalah GSoft.
  • GSoft tetap menjadi ERP dan menjadi backbone utama proses operasional dan keuangan.
  • Dashboard ini berfungsi sebagai layer monitoring dan kontrol proses, bukan pengganti ERP.

06 Detail Implementasi

6.1 CRM Module Implementation

6.1.1 State Declaration

const [isCrmModuleExpanded, setIsCrmModuleExpanded] = useState(true)

State isCrmModuleExpanded mengontrol visibilitas submenu. Default value true artinya menu terbuka saat pertama kali load.

6.1.2 Parent Menu Button

<button
  onClick={() => setIsCrmModuleExpanded(!isCrmModuleExpanded)}
  className="w-full flex items-center justify-between px-3 py-2.5
             rounded-md text-sm font-medium"
>
  <div className="flex items-center gap-3">
    <FolderKanban className="h-4 w-4" />
    <span>CRM Module</span>
  </div>
  {isCrmModuleExpanded ? <ChevronDown /> : <ChevronRight />}
</button>

6.2 Ticketing Module Implementation

6.2.1 State Declaration

const [isTicketingModuleExpanded, setIsTicketingModuleExpanded] = useState(false)

Default false karena menu ini masih placeholder. User perlu klik untuk membuka dan melihat status "Coming soon".
